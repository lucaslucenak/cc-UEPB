A partir da aplicação desenvolvida nas atividades anteriores (que já possui as páginas sobre e cadastro), forneça as seguintes páginas: 

- Cadastro de usuário: a partir da página de formulário de cadastro para um aluno, inclua a opção de professor, e esta deverá mudar algumas das opções de campos, incluindo: área de atuação e formação
- Login do usuário usando JWT Token

- Cadastro de projetos (Um projeto precisa ter, no mínimo: nome e descrição) somente a partir de um usuário professor
- Vinculação de alunos somente por um professor (terão diferentes papéis, por exemplo: estágio, júnior, pleno, sênior, master, etc.)
- Listagem de projetos e Detalhamento de um projeto com todos os integrantes, juntamente com sua função

Bônus:
- Alteração dos dados de alunos, professores e projetos

A aplicação deverá ser integrada a API fornecida abaixo (você também pode simular um mock)

O projeto pode ser feito em equipe de até 3 pessoas.

Deverá ser submetida o link da aplicação no heroku, ou na plataforma de cloud da sua preferência, juntamente com o código do GitHub.
